# Python-Daily-Expense-Tracker
Console-based Python project to record and manage daily expenses.

# Description
A console-based Python application to record and manage daily expenses. 
This project helps users track expenses, categorize them, and calculate total and category-wise expenses. 
Data is stored using file handling for persistent storage.

# Features
- Add new expenses
- View all expenses
- Categorize expenses
- Calculate total and category-wise expenses
- Persistent storage using text files

# Language & Tools
- Python
- File Handling

# Skills Learned
- Python basics: Variables, Loops, Conditionals, Functions
- Data storage using File Handling
- Dictionaries for category-wise tracking
- Logical thinking for real-world problem solving
